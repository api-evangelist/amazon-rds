# Amazon RDS (amazon-rds)

Amazon Relational Database Service (RDS) makes it easy to set up, operate, and scale a relational database in the cloud, providing cost-efficient and resizable capacity while automating time-consuming administration tasks such as hardware provisioning, database setup, patching, and backups.

## APIs

### Amazon RDS API

Core API for managing Amazon RDS database instances, clusters, snapshots, parameter groups, subnet groups, and other relational database resources across multiple database engines including MySQL, PostgreSQL, MariaDB, Oracle, SQL Server, and Amazon Aurora.

- **Human URL:** [https://aws.amazon.com/rds/](https://aws.amazon.com/rds/)
- **Base URL:** `https://rds.amazonaws.com`
